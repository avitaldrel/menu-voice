Deployment steps:
1. Push code to GitHub main branch: `git push origin main`
2. Go to https://vercel.com/new and import the menu-voice GitHub repository
3. Vercel auto-detects Next.js framework
4. BEFORE clicking Deploy, add environment variables:
   - ANTHROPIC_API_KEY (Production environment) — from https://console.anthropic.com/settings/keys
   - OPENAI_API_KEY (Production environment) — from https://platform.openai.com/api-keys
5. Click Deploy and wait for build to complete

Smoke tests on production URL (https://[project-name].vercel.app):

Test 1 — Page loads: Home page renders with MenuVoice branding and scan button, no CSP console errors.

Test 2 — Security headers: Open DevTools -> Network -> reload -> check main document response headers for Content-Security-Policy (media-src 'self' blob:), Permissions-Policy (microphone=*, camera=*), X-Content-Type-Options (nosniff), X-Frame-Options (DENY).

Test 3 — Menu photo capture: Tap Scan Your Menu, take a photo of a menu, processing completes and shows menu summary. If timeout: Edge Runtime 25s first-byte limit may have been hit.

Test 4 — Voice conversation: After menu loads, voice auto-starts with overview. Speak a question and hear a spoken TTS response. Audio plays (not blocked by CSP). Microphone works (not blocked by Permissions-Policy).

Test 5 — Usage logs: Vercel Dashboard -> project -> Logs tab shows JSON log entries with event, ip, timestamp.

Test 6 — Settings page: Navigate to /settings, page loads, can add/remove allergies.
