<objective>
Build the complete guided retake flow: detect low-quality photo extractions, show voice-guided retake UI with specific advice, and allow the user to retake or proceed with partial data.

Purpose: MENU-04 requires the app to detect unreadable photos and voice-guide the user to retake them. This is the final missing accessibility feature — without it, a blind user has no feedback when their photo was too dark or blurry.

Output: RetakeGuidance component, updated useMenuExtraction with quality detection, page.tsx wired for retake state including AppStateAnnouncer.
</objective>

  | { type: 'RETRY_CAPTURE' }

export function appReducer(state: AppState, action: AppAction): AppState;
